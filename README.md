# Real-Time Object Detection & Counting using YOLOv8

## 📌 Project Overview
Real-Time Object Detection & Counting is a Computer Vision application that uses the YOLOv8 model to detect, classify, and count objects in video streams. The system performs real-time inference by drawing bounding boxes, displaying confidence scores, and maintaining frame-wise object counts.

---

## 🎯 Problem Statement
Manual object monitoring and counting in videos is time-consuming and prone to errors. This project automates object detection and counting using the YOLOv8 deep learning model, enabling efficient real-time video analysis.

---

## 📂 Features
- 🎥 Real-Time Video Processing
- 🎯 Object Detection using YOLOv8s
- 📦 Bounding Box Visualization
- 📊 Confidence Score Display
- 🔢 Frame-wise Object Counting
- ⚡ Fast Inference Pipeline

## 🚀 Project Workflow
1. Load YOLOv8 Model
2. Read Video Frames
3. Perform Object Detection
4. Draw Bounding Boxes
5. Display Confidence Scores
6. Count Detected Objects
7. Show Processed Video Output
